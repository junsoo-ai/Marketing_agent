# ClyptAI 브랜드 가이드라인

> Version: 2026.05 · Updated for v1 launch

## 브랜드 정체성

**브랜드명:** ClyptAI (줄여서 "Clypt")
**버전:** v1 (soft launch · 2026.05)
**포지셔닝:** "The Gateway to Agent Trading"
**핵심 태그라인:** "Build your own agents. Let them trade."
**서브 태그라인:** "Your rules trigger. The agent acts."

### 핵심 thesis

> **"Retail wants an adaptive agent they own, not a static strategy they consume."**

이 한 줄이 모든 카피·포지셔닝·제품 결정의 기준점.

- *adaptive* vs static: 시장 변화에 반응하는 LLM 기반 추론
- *they own* vs they consume: 사용자가 직접 빌드·운영, 남의 전략 구독 아님

## 미션 & 비전

> **"Markets don't sleep. You can."**

리테일 트레이더가 *자기 자신의 AI 트레이딩 에이전트*를 직접 만들고 운영할 수 있게 한다. 블랙박스가 아닌, 사용자의 규칙대로 작동하고, 모든 결정을 투명하게 기록하며, 언제든 사용자가 통제할 수 있는 AI 트레이딩 인프라.

### 비전 단계

- **Act 1:** Hyperliquid 위에서 AI agent trading의 기본 layer
- **Act 2:** Multi-venue routing (Aevo, Polymarket 등) → cross-asset agent
- **Act 3:** The decision layer for retail capital across asset classes

## 브랜드 로고

### 심볼 (로고 마크)
- **형태:** 8개 모듈 유닛이 방사형(radial)으로 배치된 옥타곤(octagon) 구조
- **구성 원리:** Radial Construction · Modular Blocks · Directional Cut
  - **Radial Construction:** 중심을 기준으로 8방향 대칭으로 모듈을 배치
  - **Modular Blocks:** 각 모듈은 독립된 블록 형태 (사이 간격 = 통제된 분리)
  - **Directional Cut:** 각 블록의 한쪽 끝이 사선으로 잘려 방향성과 운동감 부여
- **의미:** 균형(balance)과 통제된 움직임(controlled movement)
- **컬러:** 단색만. 어두운 배경에서는 흰색, 밝은 배경에서는 검정

### 풀 로고 (Full Logo)
- **구성:** 심볼 + 워드마크 "Clypt"
- **워드마크 폰트:** Clypt Regular (자체 커스텀) / fallback: Public Sans
- **자간:** `-0.025em`
- **아이콘-텍스트 간격:** 심볼 너비를 `x`로 두고 `~0.4x` (랜딩 기준 12 / 31)

### 시스템 마크 (System Marks)
ClyptAI 생태계의 각 레이어를 나타내는 변형 마크.

| 마크 | 시각 특징 | 용도 |
|---|---|---|
| **Clypt Engine** (기본) | 정렬된 옥타곤 — radial construction, modular blocks, directional cut | 메인 브랜드 마크. 앱·랜딩·코어 제품 표면 |
| **Marketplace** (deprecated) | 모듈이 외측으로 확산 | v1에서 사용 안 함 (Discover/마켓플레이스 모델 deprecated) |

### 클리어 스페이스
- 심볼 너비를 `x`로 정의
- 풀 로고 기준, 사방으로 최소 `x` 만큼의 여백 확보
- 클리어 스페이스 내에는 텍스트, 다른 그래픽, 가장자리 침범 금지

### 사용 금지 (Incorrect Usage)
1. 심볼 회전 금지
2. 심볼의 시각 스타일·스트로크 변경 금지 (아웃라인 변형 불가)
3. 심볼 비율 왜곡·변형 금지
4. 클리어 스페이스 침범 금지
5. 풀 로고 가로 압축 금지
6. 심볼-텍스트 간격 임의 확장 금지
7. 심볼을 워드마크 위에 적층(stack) 금지 (가로 정렬만)
8. 심볼에 색상 적용 금지 (단색 흰색/검정만)

## 타이포그래피

> ⚠️ **확인 필요:** Clypt 커스텀 패밀리(Bold / Bold Stencil / Regular)가 실제 deploy되어 있는지, 또는 Inter 같은 시스템 폰트로 fallback되는지 디자인 팀과 확인 필요.

### 폰트 패밀리

| 역할 | 폰트 | 케이스 | Kerning | Line Height | 메인 용도 |
|---|---|---|---|---|---|
| Display / Headline | **Clypt Bold** | Original | 0% | 110% | 히어로 섹션 헤드라인, 랜딩 메인 카피 |
| Display / 대형 숫자 | **Clypt Bold Stencil** | Original | 0% | 110% | 큰 폰트 사이즈의 숫자 (랜딩 KPI, 대형 메트릭) |
| Subtitle | **Clypt Regular** | Original | 0% | 110% | 섹션 서브타이틀, 워드마크 |
| Body | **Public Sans Regular** | Original | -1% | 120% | 모든 본문 카피 |
| CTA / Caption | **Public Sans Regular** | All Caps | 0% | 120% | CTA 버튼, 내비게이션, 캡션 |
| 코드 / 정밀 숫자 | **JetBrains Mono** | Original | 0% | — | 가격, 식별자, 코드, 표 안의 수치 |

> **Clypt 커스텀 패밀리(Bold / Bold Stencil / Regular)** 는 브랜드 표면(랜딩, 마케팅, 헤더)의 디스플레이 타이포 전용. 본문은 Public Sans, 숫자는 JetBrains Mono 원칙 유지.

### 타입 스케일
| 클래스 | 크기 | 웨이트 | 폰트 | 용도 |
|---|---|---|---|---|
| `.h-display` | 56px | Bold | Clypt Bold | 랜딩 히어로 |
| `.h1` | 40px | 400 | Public Sans / Clypt Regular | 페이지 제목 |
| `.h2` | 32px / 40px | 500 | Public Sans / Clypt Regular | 섹션 제목 |
| `.h3` | 24px / 30px | 500 | Public Sans | 카드 제목 |
| `.h4` | 20px / 28px | 500 | Public Sans | 서브 제목 |
| `.body-lg` | 18px | 400 | Public Sans | 주요 본문 |
| `.body` | 16px | 400 | Public Sans | 일반 본문 |
| `.body-sm` | 14px | 400 | Public Sans | 보조 본문 |
| `.caption` | 12px | 500 | Public Sans (All Caps) | 캡션 |
| `.label-eyebrow` | 12px | 600 | Public Sans (All Caps) | 섹션 아이브로우 레이블 |

### 타이포 규칙
- **숫자는 반드시 JetBrains Mono** + `font-variant-numeric: tabular-nums`
- **대형 디스플레이 숫자(랜딩 KPI 등)** 는 `Clypt Bold Stencil`로 표현 가능
- **Eyebrow / CTA 레이블:** All Caps + `letter-spacing: 0.08–0.14em`
- **헤딩 자간:** `-0.01em` ~ `-0.025em`
- **Body 자간:** `-0.01em` (Public Sans Regular의 -1% 가이드 반영)
- **디스플레이 행간:** Clypt 패밀리 사용 시 line-height 110%

## 브랜드 보이스 & 톤

### 글로벌 카피 원칙

1. **투명성:** 모든 결정·실행·이유는 기록되고 노출됨
2. **사용자 통제:** 본인의 규칙대로만 작동. 일시정지 언제든 가능
3. **신뢰:** 온체인 검증, 비수탁
4. **단순함:** 복잡한 트레이딩 jargon 최소화

### 영문 카피 원칙

**금지**
- Em-dash (—) 사용 금지 — AI 생성 텍스트 시그널로 인식
- 과장 표현 ("revolutionary", "industry-first")
- 손실 강조형 ("you'll lose money if...")

**선호**
- 두 박자 리듬 ("X. Y." 패턴)
- 직설적 명령조 ("Watch.", "Trade.", "Sleep.")
- 구체적 수치 + 출처 ($80B/yr global hedge fund fees 등)

### 한글 카피 원칙 (신규)

**대명사**
- **"본인" 사용 우선**, "당신" 지양
- 또는 주어 생략으로 자연스러움

**기호**
- Em-dash (—) 금지 → 마침표, 쉼표, 또는 가운뎃점(·)으로 대체
- 슬래시(/)는 가능하면 가운뎃점(·)으로 ("매수/롱" → "매수 · 롱")

**문장**
- 완전한 문장 선호 (telegraphic 단답형 지양)
- "무료. 카드 없음." → "회원가입 없이 무료로 시작하세요"

**브랜드 명사**
- 영문 유지: ClyptAI, Telegram, Privy, Hyperliquid, Builder code
- 한국어 음차 지양 ("하이퍼리퀴드" 보다 "Hyperliquid" 선호)

**Hybrid 자연스러움**
- 한글-영문 mix 자연스럽게
- 직역체 회피 ("activate the agent" → "에이전트 활성화" 가능. "에이전트를 활성화시키세요" 어색)

### Five Moments 카피 톤 (앱 내 일관성)

| 단계 | 영문 헤딩 | 한글 헤딩 |
|------|---------|---------|
| Watch | Your agent watches. You don't have to. | 에이전트가 봅니다. 본인은 쉬세요. |
| Signal | Your rules trigger. The agent acts. | 본인의 규칙이 발동. 에이전트가 실행합니다. |
| Wait | No setup, no entry. The agent holds. | 조건이 없으면, 진입도 없습니다. 에이전트는 기다려요. |
| Enter | You see the receipt. Not the click. | 체결 결과만 보세요. 클릭은 에이전트가. |
| Sleep | Markets don't sleep. You can. | 시장은 잠들지 않아요. 본인은 자도 됩니다. |

→ 이 5개 라인의 톤·리듬을 다른 카피에서도 reference로 사용.

### Auto-execution Honesty (중요)

v1은 auto-execution 단독 모드. 카피에서 *"signal-and-approve"* 톤 금지.

- ❌ "Conditions met. Confirm to execute."
- ✅ "Conditions met. The agent executes."
- ❌ "조건 충족 시 알림. 승인하거나 건너뛰세요."
- ✅ "조건 충족 시 자동 실행. 모든 동작은 기록되고 일시정지 가능."

대신 **사용자 통제는 사전 설정 + 사후 일시정지로 일관되게 표현**.

### Pivot Story 처리 (중요)

외부 카피에 *"pivot"* 단어 사용 금지. 대신 사용:
- "evolved" / "sharpened"
- "based on user feedback"
- "We landed on..."
- "Our thesis got clearer"

내부 문서·deck에선 솔직하게 처리 가능.

## 신뢰 시그널 (Trust Signals)

마케팅 자산에 일관되게 노출되어야 하는 신뢰 요소:

- **Built on Hyperliquid via Builder Code program**
- **Non-custodial via Privy** (Your keys, your funds)
- **Member of Anthropic Claude Partner Network**
- **All trades verified on-chain**
- **No silent moves** (모든 거래는 로그됨)
- **Pause anytime** (언제든 일시정지 가능)

## 사용 금지 카피 / Anti-Patterns

다음 표현은 ClyptAI 브랜드 카피에서 사용 금지:

| 금지 표현 | 이유 | 대안 |
|---------|------|------|
| "Revolutionary AI trading" | 과장 | "AI agents that trade for you" |
| "Guaranteed returns" | 법적 위험 | "Your rules, your results" |
| "Beat the market" | 과장 + 부정확 | "Stay in the market on your terms" |
| Em-dash (—) | AI 시그널 | 마침표, 쉼표, 가운뎃점 |
| "You vs. the agent" | 적대적 톤 | "You set the rules, the agent acts" |
| "당신" (한글) | 톤 가이드 | "본인" 또는 주어 생략 |
| "Pivoted" (외부) | 부정적 시그널 | "Evolved" / "Sharpened" |
